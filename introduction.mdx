Tintero is a set of smart contracts that allow lending ERC-20 tokens like USDC in exchange for tokenized Real World Assets (RWA) generated by [Plumaa ID](https://docs.plumaa.id) as legally-binding obligations.

The protocol establishes rules through which investors can provide liquidity for assets that generate cash-flow from debt (amongst other negotiables). Examples of these are loans, leases, receivables, and other financial instruments represented by ERC-721 tokens.
